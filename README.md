# The Ball Pit
**NOTE:** This repository is solely designed to maintain the source code used for deployment.

## Purpose
The Ball Pit will serve as a community catalog for items related to Pizza Time Theatre, ShowBiz Pizza Place, and Chuck E. Cheese’s, providing direct and to-the-point information regarding each brand, its characters, store locations and concepts, stage shows, animatronics, walkaround costumes, showtapes, showtape formats, and showtape transcriptions.

The underlying system will be designed to encourage community contributions that expand the database, with guardrails to keep the information clean, so that the catalog can coexist with other community resources as a primary source for quick research and reference.

## Feature Wishlist
### Purposeful Data
The catalog will be designed so that every entry connects to each other. Users will be able to quickly find the information they are looking for by using content trees, category-based navigation, or through basic title and property search. The editor for each entry will differ based on the type of item it is, using existing entries to populate dropdown menus or other form elements.

### Community Notes & User Enablement
Each entry in the catalog will have a community notes feed where users can have on-topic conversations with other users. Notes can be upvoted or downvoted so that low quality notes can be marked for review and hidden by trusted users. Any user can contribute an edit, but edits must be approved by a trusted user. Those same trusted users will be able to make edits without review and approval from other trusted users.

### Quick Sharing & Discoverability
URL slugs for catalog entries will be generated based on their names. For each entry, prior URLs are archived so that name changes do not break existing links. If a new page is created that has the same URL as an archived link, the user will be given the option to choose their destination. Trusted users can manage these conflicts in a separate interface.
